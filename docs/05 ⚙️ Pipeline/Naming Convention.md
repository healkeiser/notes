## Assets

```
<assetName><identifier>
```

```
characterFunnyA
```

## Textures

```
<TYPE>_<textureName>_<SIZE>.<UDIM>.<FRAME>.exr
```

```
DIF_chestPlate_4K.1001.1001.exr
```

### Valid Types

| Type   | Long Name         |
| ----:  | ----------------- |
| `DIF`  | Diffuse           |
| `RGH`  | Roughness         |
| `MTL`  | Metalness         |
| `NML`  | Normal            |
| `DSP`  | Displacement      |
| `AO`   | Ambient Occlusion |
